# 创建数组的方式


```swift
/**
 创建数组的方式
 */

// 1.字面量创建
let arr = [1,2,3,4,5]
// 创建空数组时，需要指定数组类型
let arr1: [String] = []

// 2.初始化器创建
let arr2 = [String]()
let arr3 = Array<String>()
let arr4 = Array<String>(repeating: "s",count: 5)
let person = ["zhangsan": 10, "lisi": 12, "wangwu": 14]
let arr5 = [String](person.keys)
print(arr5)
```
